Безопасность базы данных конфигурации
=====================================

В базе данных конфигурации хранятся открытый и закрытый ключи, контролирующие доступ ко всему содержимому, которое лицензируется установкой службы управления правами, поэтому очень важно, чтобы база данных была надежно защищена. Чтобы обеспечить надежность защиты базы данных, необходимо ограничить физический доступ к серверам, на которых размещена база данных конфигурации, а также убедиться, что все компоненты программного обеспечения настроены таким образом, чтобы доступ к нему был разрешен только для уполномоченного персонала. По умолчанию служба управления правами настраивает разрешения базы данных конфигурации с обеспечением надежной защиты. Дополнительные сведения о дополнительных шагах, предпринимаемых для защиты базы данных конфигурации, можно найти в разделе “[Безопасность баз данных, используемых службой управления правами](https://technet.microsoft.com/65802f9a-81bc-4398-968a-00c9b1dca2fa)”данного документа.
