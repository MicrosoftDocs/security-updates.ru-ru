Security Bulletin

Бюллетень по безопасности (Майкрософт) MS12-026 - Важное
========================================================

Уязвимости в Forefront Unified Access Gateway (UAG) могут привести к раскрытию информации (2663860)
---------------------------------------------------------------------------------------------------

Дата публикации: 10 апреля 2012 г. | Дата обновления: 18 апреля 2012 г.

**Версия:** 1.1

### Общие сведения

#### Аннотация

Данное обновление для системы безопасности устраняет две обнаруженных пользователями уязвимости в Microsoft Forefront Unified Access Gateway (UAG). Наиболее серьезная из этих уязвимостей делает возможным раскрытие информации, если злоумышленник отправит специально созданный запрос в сервер UAG.

Данное обновление безопасности имеет уровень "существенный" для Microsoft Forefront Unified Access Gateway 2010 с пакетом обновления 1 (SP1) и Microsoft Forefront Unified Access Gateway 2010 с обновлением 1 пакета обновления 1 (SP1 Update 1). Дополнительные сведения см. в подразделе **Подвержены и не подвержены уязвимости** далее в этом разделе.

Данное обновление устраняет уязвимости, изменяя код UAG для выполнения дополнительной проверки перед перенаправлением пользователя на другой веб-сайт, а также измененяя параметры привязки по умолчанию сервера UAG для запрета доступа к внутренним ресурсам без фильтрации. Дополнительные сведения об этих уязвимостях см. в подразделе ответов на часто задаваемые вопросы для каждой уязвимости, упомянутой в разделе **Сведения об уязвимостях**.

**Рекомендация.** Корпорация Майкрософт рекомендует установить это обновление при первой возможности.

См. также раздел **Руководство и средства по диагностике и развертыванию** этого бюллетеня.

**Известные проблемы.**[В статье 2663860 базы знаний Майкрософт](http://support.microsoft.com/kb/2663860) описаны известные на данный момент проблемы, с которыми пользователи могут столкнуться при установке этого обновления для системы безопасности. В этой статье также приводятся решения, рекомендованные для устранения этих проблем. Для случаев, когда известные проблемы и методы их решения относятся к определенным выпускам программного обеспечения, предоставляются ссылки на соответствующие статьи.

Уязвимость перенаправления HTTP вслепую в UAG (CVE-2012-0146)  
-------------------------------------------------------------
  
<span></span>
Существующая уязвимость Forefront Unified Access Gateway (UAG), которая делает возможным спуфинг, может привести к раскрытию информации. Эта уязвимость допускает спуфинг путем перенаправления веб-трафика, предназначенного для сервера UAG, на вредоносный веб-сайт. Злоумышленник может воспользоваться этой уязвимостью, отправив пользователю сервера UAG ссылку со специально созданным URL-адресом и убедив пользователя щелкнуть эту ссылку. Когда прошедший проверку пользователь UAG щелкнет эту ссылку, сеанс браузера этого пользователя может быть перенаправлен на вредоносный веб-сайт, специально созданный для подмены подлинного веб-интерфейса UAG. Таким образом злоумышленник может обмануть пользователя и получить конфиденциальную информацию (например, учетные данные пользователя).
  
Чтобы просмотреть стандартную запись об этой уязвимости в списке "Common Vulnerabilities and Exposures", см. [CVE-2012-0146](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2012-0146).
  
#### Смягчающие факторы для уязвимости, связанной с перенаправлением HTTP вслепую в UAG (CVE-2012-0146)
  
К смягчающим факторам относятся параметр, стандартная конфигурация или общие рекомендации, которые при использовании по умолчанию могут снизить опасность использования уязвимости. Следующие смягчающие факторы могут снизить опасность использования уязвимости.
  
-   Злоумышленник не может заставить пользователей посетить вредоносный веб-сайт. Вместо этого злоумышленнику потребуется убедить пользователя щелкнуть ссылку со специально созданным URL-адресом, которая перенаправляет пользователя на такой веб-сайт, или щелкнуть ссылку в сообщении электронной почты или программы обмена мгновенными сообщениями, которая перенаправляет пользователя на веб-сайт злоумышленника.
  
#### Временные решения для уязвимости, связанной с перенаправлением HTTP вслепую в UAG (CVE-2012-0146)
  
Корпорацией Майкрософт не найдено временных решений для устранения этой уязвимости.
  
#### Часто задаваемые вопросы об уязвимости, связанной с перенаправлением HTTP вслепую в UAG (CVE-2012-0146)
  
**Какова область воздействия этой уязвимости?**  
На серверах Forefront UAG существует уязвимость, делающая возможным спуфинг. Воспользовавшись этой уязвимостью, злоумышленник может подменить подлинный веб-интерфейс UAG.
  
**В чем причина уязвимости?**  
Уязвимость возникает, когда веб-интерфейсу UAG не удается проверить и подтвердить перенаправление на внешний веб-сайт.
  
**Что может сделать злоумышленник, который воспользуется этой уязвимостью?**  
Злоумышленник может перенаправить прошедшего проверку пользователя UAG на вредоносную веб-страницу и убедить его ввести свое имя пользователя, пароль или другую личную информацию.
  
**Каким образом злоумышленник может воспользоваться этой** **уязвимостью?**  
Злоумышленнику убеждает пользователя щелкнуть ссылку UAG со специально созданным URL-адресом в сообщении электронной почты, программы обмена мгновенными сообщениями или другой программы доставки. Эта ссылка должна перенаправить сеанс браузера пользователя на произвольный сайт, контролируемый злоумышленником, в то время когда пользователь считает, что просматривает подлинный веб-сайт.
  
**Какие системы в первую очередь подвержены риску?**  
Серверы, на которых выполняется Microsoft Unified Access Gateway, подвержены риску атаки с использованием этой уязвимости.
  
**Как действует обновление?**  
Данное обновление устраняет уязвимость, изменяя код UAG для выполнения дополнительной проверки перед перенаправлением пользователя на другой веб-сайт.
  
**Была ли эта уязвимость опубликована до выпуска этого бюллетеня безопасности?**  
Нет. Корпорация Майкрософт получила сведения об этой уязвимости из надежных источников.
  
**На момент выпуска этого бюллетеня получала ли корпорация Майкрософт сообщения об использовании этой уязвимости?**  
Нет. Корпорация Майкрософт не получала никакой информации, указывающей на то, что до выпуска этого бюллетеня по безопасности данная уязвимость была открыто использована для организации атак на пользовательские компьютеры.
  
Уязвимость, связанная с доступом к веб-сайту по умолчанию в UAG (CVE-2012-0147)  
-------------------------------------------------------------------------------
  
<span></span>
Уязвимость в Microsoft Unified Access Gateway (UAG) позволяет неавторизованному пользователю из внешней сети получить доступ к веб-сайту по умолчанию на сервере Microsoft UAG.
  
Чтобы просмотреть стандартную запись об этой уязвимости в списке "Common Vulnerabilities and Exposures", см. [CVE-2012-0147](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2012-0147).
  
#### Смягчающие факторы для уязвимости, связанной с доступом к веб-сайту по умолчанию в UAG (CVE-2012-0147)
  
Корпорация Майкрософт не обнаружила смягчающих факторов для данной уязвимости.
  
#### Временные решения для уязвимости, связанной с доступом к веб-сайту по умолчанию в UAG (CVE-2012-0147)
  
Корпорацией Майкрософт не найдено временных решений для устранения этой уязвимости.
  
#### Часто задаваемые вопросы об уязвимости, связанной с доступом к веб-сайту по умолчанию в UAG (CVE-2012-0147)
  
**Какова область воздействия этой уязвимости?**  
Это уязвимость, делающая возможным раскрытие информации.
  
**В чем причина уязвимости?**  
Эта уязвимость вызвана неверной конфигурацией веб-сайта по умолчанию на сервере UAG, которая допускает доступ из внешней сети к некоторому содержимому на этом веб-сайте.
  
**Что может сделать злоумышленник, который воспользуется этой уязвимостью?**  
Злоумышленник, успешно воспользовавшийся этой уязвимостью, может просмотреть защищенные ресурсы на сервере.
  
**Каким образом злоумышленник может воспользоваться этой уязвимостью?**  
Удаленный непроверенный злоумышленник может воспользоваться данной уязвимостью, отправив специально созданный HTTPS-запрос на сервер UAG. Этот специально созданный HTTPS-запрос может позволить злоумышленнику получить доступ к закрытым ресурсам через веб-сайт по умолчанию на сервере UAG.
  
**Какие системы в первую очередь подвержены риску?**  
Серверы, на которых выполняется Microsoft Unified Access Gateway, подвержены риску атаки с использованием этой уязвимости.
  
**Как действует обновление?**  
Данное обновление устраняет уязвимость, изменяя параметры привязки по умолчанию сервера UAG для запрета доступа к внутренним ресурсам без фильтрации.
  
**Была ли эта уязвимость опубликована до выпуска этого бюллетеня безопасности?**  
Нет. Корпорация Майкрософт получила сведения об этой уязвимости из надежных источников.
  
**На момент выпуска этого бюллетеня получала ли корпорация Майкрософт сообщения об использовании этой уязвимости?**

### Прочие сведения

#### Программа Microsoft Active Protections Program (MAPP)
