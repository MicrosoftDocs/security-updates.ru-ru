Security Bulletin

Бюллетень по безопасности (Майкрософт) MS06-075 - Важное
========================================================

Уязвимости в системе Windows могут привести к повышению уровня прав доступа (926255)
------------------------------------------------------------------------------------

Дата публикации: 12 декабря 2006 г.

#### Заключение

**Для кого предназначен этот документ:** для пользователей Microsoft Windows

**Воздействие уязвимости:** Повышение уровня прав доступа

**Степень опасности:** Важно

**Рекомендация:** Пользователи должны как можно скорее установить это обновление.

**Замена обновлений безопасности:** Отсутствуют.

**Предупреждения.** Отсутствуют.

**Проверенное программное обеспечение и адрес загрузки соответствующих обновлений безопасности.**

**Подвержены уязвимости:**

-   Microsoft Windows XP с пакетом обновления 2 (SP2) — [загрузить обновление](http://www.microsoft.com/downloads/details.aspx?familyid=966704b5-1a7e-4110-9694-844706a52db7)
-   Система Microsoft Windows Server 2003: [загрузить обновление](http://www.microsoft.com/downloads/details.aspx?familyid=5ea314a2-d76a-46f9-853b-15ff03f8ad95)
-   Система Microsoft Windows Server 2003 для платформы Itanium: [загрузить обновление](http://www.microsoft.com/downloads/details.aspx?familyid=7bceaa11-f655-4e3c-a588-5c49097e970b)

**Не подвержены уязвимости:**

-   Windows 2000 с пакетом обновления 4 (SP4)
-   Microsoft Windows XP Professional (версия x64)
-   Microsoft Windows Server 2003 с пакетом обновления 1 (SP1)
-   Microsoft Windows Server 2003 с пакетом обновления 1 для платформы Itanium
-   Microsoft Windows Server 2003 (версия x64)
-   ОС Windows Vista

Корпорация Майкрософт проверила на наличие уязвимости продукты перечисленных выше версий. Другие версии больше не поддерживаются либо не являются уязвимыми. Сведения об уязвимости  
----------------------
  
<span></span>
#### Уязвимость повреждения манифеста файла — CVE-2006-5585:
  
Уязвимость [несанкционированного получения прав](http://go.microsoft.com/fwlink/?linkid=21142) имеется в способе запуска системой Microsoft Windows приложений со специально сконструированными манифестами файла. Данная уязвимость может позволить зарегистрировавшемуся пользователю захватить полный контроль над системой.
  
#### Факторы, снижающие опасность уязвимости повреждения манифеста файла — CVE-2006-5585.
  
-   Чтобы воспользоваться данной уязвимостью, злоумышленник должен обладать действительными учетными данными и возможностью локального входа в систему. Данной уязвимостью не могут воспользоваться удаленные или анонимные пользователи.
  
#### Методы обхода уязвимости повреждения манифеста файла — CVE-2006-5585.
  
Для данной уязвимости методы обхода отсутствуют.
  
#### Вопросы и ответы об уязвимости повреждения манифеста файла — CVE-2006-5585.
  
**Какова область воздействия данной уязвимости?**  
Эта уязвимость может приводить к [повышению уровня прав](http://go.microsoft.com/fwlink/?linkid=21142). Воспользовавшись этой уязвимостью, злоумышленник может захватить полный контроль над системой, Таким образом, злоумышленник сможет устанавливать программы, просматривать, изменять и уничтожать данные или создавать новые учетные записи с неограниченными полномочиями. Чтобы воспользоваться уязвимостью, злоумышленник должен зарегистрироваться в локальной системе и запустить программу.
  
**Чем вызвана данная уязвимость?**  
Неправильной обработкой манифеста файлов и управление ими клиент-серверной подсистемой времени выполнения.
  
**Что такое клиент-серверная подсистема времени выполнения (CSRSS)?**  
Подсистема Client Server Run Time Subsystem (**CSRSS**) представляет собой часть подсистемы Win32, обслуживающую работу в пользовательском режиме (в то время как Win32.sys является частью, обслуживающей работу в привилегированном режиме). Аббревиатура Csrss означает *клиент-серверная подсистема времени выполнения*. Это очень важная подсистема, которая должна функционировать постоянно. Подсистема Csrss отвечает за окна консолей, создание и удаление потоков.
  
**Что может сделать злоумышленник, воспользовавшись этой уязвимостью?**  
Воспользовавшись этой уязвимостью, злоумышленник может получить полный контроль над системой.
  
**Кто может воспользоваться этой уязвимостью?**  
Чтобы воспользоваться уязвимостью, злоумышленник должен зарегистрироваться в локальной системе и запустить программу.
  
**Каким образом злоумышленник может воспользоваться уязвимостью?**  
Чтобы воспользоваться данной уязвимостью, злоумышленник должен вначале зарегистрироваться в системе. Затем злоумышленник может запустить специальное приложение, которое способно воспользоваться уязвимостью и таким образом установить полный контроль над системой.
  
**Какие системы в первую очередь подвергаются риску?** В первую очередь уязвимы рабочие станции и серверы терминалов. Серверы могут быть подвержены большему риску, если администраторы позволяют пользователям входить в систему на сервере и запускать программы. Однако общепринятой практикой настоятельно рекомендуется не предоставлять пользователям таких полномочий.
  
**Можно ли воспользоваться данной уязвимостью через Интернет?**  
Нет. Злоумышленник должен зарегистрироваться в конкретной системе, на которую нацелена атака. Используя данную уязвимость, злоумышленник не может загрузить и запустить программу удаленно.
  
**Как действует обновление?**  
Обновление устраняет данную уязвимость посредством изменения способа проверки встроенных манифестов файла клиент-серверной подсистемой времени выполнения перед передачей данных в выделенный буфер.
  
**Было ли объявлено об уязвимости до выпуска этого бюллетеня безопасности?**  
Нет. О данной уязвимости не было открыто объявлено до выпуска бюллетеня.
  
**Как данная уязвимость связана со статьей 921337 базы знаний Майкрософт?**  
[В статье 921337 базы знаний Майкрософт](http://support.microsoft.com/kb/921337/en-us) описывается связанная с данной уязвимостью проблема манифеста файла, которая потенциально может вызвать перезагрузку системы. В процессе исследования этой проблемы корпорация Майкрософт обнаружила потенциальную уязвимость системы безопасности, для устранения которой выпущено данное обновление.
