Security Advisory

Советы по безопасности (Microsoft) 943411
=========================================

Обновление для укрепления защиты боковой панели Windows
-------------------------------------------------------

Дата публикации: 8 января 2008 г.

Для поддерживаемых в данный момент выпусков Windows Vista доступно обновление. Это обновление, предназначенное для укрепления защиты боковой панели Windows, включает в ней блокировку запуска мини-приложений. Дополнительные сведения об установке этого обновления см. в [статье 943411 базы знаний Майкрософт](http://support.microsoft.com/kb/943411). Сведения о том, как защита боковой панели Windows позволяет блокировать в ней запуск установленных мини-приложений, см. в [статье 941411 базы знаний Майкрософт](http://support.microsoft.com/kb/941411).

В документе [Шесть советов по безопасности при использовании мини-приложений в Windows Vista](http://www.microsoft.com/protect/yourself/downloads/gadgets.mspx) (на английском языке) содержатся рекомендации по безопасным загрузке и использованию мини-приложений. В документе MSDN [Безопасность мини-приложений для боковой панели Windows](http://msdn2.microsoft.com/en-us/library/bb508510.aspx) (на английском языке) содержатся сведения о модели безопасности боковой панели Windows. В документе MSDN [Проверка мини-приложений](http://msdn2.microsoft.com/en-us/library/bb498012.aspx) (на английском языке) содержатся рекомендации по защищенному программированию, которые необходимо соблюдать при создании мини-приложений.

Вопросы и ответы
----------------

<span></span>
**Какова область действия данных рекомендаций? **
Объявление о доступности обновления для боковой панели Windows и предоставление сведений о его назначении.

**Что такое мини-приложения? **
Назначение мини-приложений — предоставлять пользователям сведения и программы. Система Windows Vista обрабатывает их так же, как и любой другой исполняемый код. Они создаются с помощью HTML-кода и сценариев, но этот HTML-код, в отличие от веб-страниц, размещен не на произвольном удаленном сервере. HTML-содержимое мини-приложения сначала загружается как часть пакета ресурсов и файлов конфигурации, а затем запускается на локальном компьютере. Этот процесс загрузки одинаков для всех приложений (EXE-файлов), загружаемых из Интернета.

**Что такое защита боковой панели Windows? **
Защита боковой панели Windows позволяет системе Windows Vista блокировать запуск мини-приложений в этой панели, что позволяет защититься от возможных уязвимостей в их системе безопасности.

**Почему корпорация Майкрософт выпускает это обновление? **
Корпорация Майкрософт выпускает это обновление, чтобы обеспечить дополнительную защиту боковой панели Windows. Оно позволяет защите боковой панели Windows блокировать потенциально уязвимые мини-приложения с целью снижения опасности, которая может возникнуть при их использовании.

**Будут ли запускаться мини-приложения в боковой панели после установки этого обновления? **
Да. Мини-приложения по-прежнему будут запускаться. Обновление не блокирует запуск определенных мини-приложений в боковой панели Windows. Оно позволяет защитить боковую панель Windows от потенциальных угроз для системы безопасности, исходящих от мини-приложений.

**Требует ли эта уязвимость безопасности выпуска обновления корпорацией Майкрософт? **
Нет. Это обновление расширяет возможности боковой панели, но не затрагивает существующие в ней или в доступных мини-приложениях уязвимости системы безопасности.

**В каком разделе веб-узла Центра обновления Windows будет представлено данное обновление?**  
На веб-узле Центра обновления Windows это обновление имеет высокий приоритет. Пользователи, которые еще не получили это обновление и работают с перечисленными выше программами, могут найти его в списке высокоприоритетных обновлений на веб-узле Центра обновления Windows.

**Нужно ли устанавливать данное обновление, если операционная система Windows регулярно обновлялась? **
Да. Несмотря на то что это обновление не устраняет никаких критических уязвимостей системы безопасности, оно повышает надежность и устойчивость боковой панели Windows.

**Будет ли данное обновление распространяться через службу автоматического обновления? **
Да, для программного обеспечения, перечисленного выше, оно распространяется через службу автоматического обновления.
