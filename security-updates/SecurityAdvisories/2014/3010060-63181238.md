Microsoft Security Advisory 3010060
===================================

Vulnerability in Microsoft OLE Could Allow Remote Code Execution
----------------------------------------------------------------

Published: October 21, 2014 | Updated: November 11, 2014

**Version:** 2.0

General Information
-------------------

### Executive Summary

Microsoft has completed the investigation into a public report of a vulnerability. We have issued Microsoft Security Bulletin [MS14-064](http://go.microsoft.com/fwlink/?linkid=518106) to address this issue. For more information about this issue, including download links for an available security update, please review the security bulletin. The vulnerability addressed is the Windows OLE Remote Code Execution Vulnerability - [CVE-2014-6352](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2014-6352).Â
